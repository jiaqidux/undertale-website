# Undertale Website
 
A personal website built for CS50's Week 8 assignment, themed around Undertale. 
Built entirely with static HTML, CSS, and JavaScript.
 
## Live Site
 
[View the deployed site](https://jiaqidux.github.io/undertale-website/) *(hosted via GitHub Pages)*
 
## How to run it locally
 
**Option 1: Just open it directly**
Double-click `index.html`, or open it via your browser:
```
open index.html
```
(On Windows: just double-click the file in File Explorer.)
 
**Option 2: VS Code Live Server extension**
If you're using VS Code, install the "Live Server" extension, right-click `index.html`, and choose "Open with Live Server."
